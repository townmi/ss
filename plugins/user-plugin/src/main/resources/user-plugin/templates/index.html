查看 API 文档

API 接口文档

获取所有用户

GET /users

返回系统中所有用户的列表

获取单个用户

GET /users/:id

根据用户 ID 获取特定用户的详细信息

创建新用户

POST /users

创建新用户，需要在请求体中提供用户信息

页面路由

用户列表页面

GET /page/users

展示所有用户的网页界面

用户详情页面

GET /page/users/:id

展示特定用户详细信息的网页界面

插件主页

GET /page/

用户管理插件的主页（当前页面）
