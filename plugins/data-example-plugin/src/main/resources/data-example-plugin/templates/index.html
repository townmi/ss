核心特性

  • 类型安全：通过泛型确保编译时类型检查
  • Repository 模式：提供统一的数据访问接口
  • 灵活的查询：支持多种查询条件和组合
  • 自动时间戳：自动管理创建和更新时间
  • 批量操作：支持批量创建和删除
  • 分页支持：内置分页和排序功能

快速开始：定义实体

public class User extends Entity {
    private String username;
    private String email;
    private String role;
    private boolean active;
    
    // getters and setters...
}

使用 Repository 进行数据操作

// 获取仓库
Repository<User> userRepo = dataService.getRepository(Collections.USERS, User.class);

// 创建用户
User user = new User();
user.setUsername("john_doe");
user.setEmail("john@example.com");
User saved = userRepo.save(user);

// 查询活跃的管理员
List<User> admins = userRepo.findBy(
    QueryCriteria.<User>create()
        .eq("role", "admin")
        .eq("active", true)
);
