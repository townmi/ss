API 接口文档

获取用户权限

GET /auth/permissions/:userId

获取指定用户的所有权限列表

授予权限

POST /auth/permissions/:userId

为用户授予新的权限，请求体需包含 permission 字段

撤销权限

DELETE /auth/permissions/:userId/:permission

撤销用户的指定权限

检查权限

GET /auth/check/:userId/:permission

检查用户是否拥有指定权限
